PDMS User Interface

The PDMS web application consists of four interfaces: the GEE Data Downloader, MAP, Plots, and Dashboard. Each of these interfaces is described below.

GEE Data Downloader

Click GEE Data Downloader tab to display the GEE Data Downloader Tool. Enter value of parameters to generate GEE Data.

Map (Main Page)

Click Map tab to display the following window.

Plot List

Survey sites registered via the web application or results sent from the field will be displayed here.

Click each item on the list to be redirected to a detailed information of survey site.
