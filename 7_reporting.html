Reporting and Dashboard

In the PDMS Web application, you can download survey results in Excel or report formats for reporting and analysis of deforestation in your target area. The application also provides dashboard that visualize survey results and provide necessary information to address deforestation.

Export of the Results of Field Survey

Download all survey results for the target province in Excel format

  1. Select [Plot List] tab in the PDMS Web application, then click “Export Excel File” button at the bottom of the screen. The Excel file is downloaded as output.xlsx.

Download all deforestation locations in the target province in KML format

  1. Select [Plot List] tab in the PDMS Web application, then click KML download button at the bottom of the screen. The KML file is downloaded as kml_list.kml and you can view in Google Earth.

Download individual survey results in report format

  1. Select [plot List] tab in the PDMS Web application, then double-click the data of your interest from the list of Plot Data.

  2. Click the Export Survey Report (Excel) button to start downloading the selected data of your interest as reporting format.

(6) Click the Download KML file (Field Measurement) button to start downloading the selected data in KML format. The KML file will be downloaded as pt_(Plot ID).kml and can be viewed in Google Earth.

Dashboard

  1. Select [Dashboard] tab in the PDMS Web application

  1. You can see the Dashboard showing the result of data analysis in your target province

  1. If you want to download the Dashboard as PDF format, click Share, then click Download in the upper right.
